Random.JS

Random.JS is JavaScript random library.
On github: Random.JS

random.getrandbits(k)

Returns random integer with k random bits.

random.random()

Returns random float in range [0.0, 1.0].

random.randint(a, b)

Returns random integer in range [a, b].

random.randrange(x)

Returns random integer in range [0, a].

random.randrange(x, y)

Returns random integer in range [a, b].

random.randrange(x, y, z)

Returns random integer in range [a, b] with step z.

random.choice(array)

Returns random element from array.

random.shuffle(array)

Shuffles array.

random.uniform(a, b)

Returns random float in range [a, b].
